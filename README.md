# Vidyavya — Career Launch

Accelerate your tech career with Vidyavya. Industry-led 6-month career engineering programs in Data Engineering and AI/ML.

## Overview

This repository contains the front-end application built for Vidyavya Career Launch utilizing React, TypeScript, Vite, and tailwindcss.

## Getting Started

```sh
# Install dependencies
npm i

# Start the development server
npm run dev
```
